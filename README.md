## Featured Work

🔐 **API Testing — Expert Level**
Exploited Server-Side Parameter Pollution in a REST URL — chained path 
traversal + API version navigation + JavaScript source analysis to extract 
admin password reset token.

💡 **Business Logic — Expert Level**
UTF-7 parser differential attack — exploited encoding inconsistency between 
frontend validation and backend processing. 5 hours solo.

💉 **XXE — Expert Level**
Local DTD repurposing attack — triggered FileNotFoundException to exfiltrate 
/etc/passwd without Burp Collaborator.

🛡️ **CSRF — All 11 Practitioner Labs**
Full coverage including SameSite bypass, Referer bypass, OAuth cookie refresh 
exploitation and CRLF injection.

🌐 **SSRF — Complete Series**
Internal network pivoting, AWS IAM credential extraction via EC2 metadata 
and open redirect chains.

🤖 **Gandalf Prompt Injection — Top 8% Worldwide**
Cracked all 8 levels including Gandalf the White using acrostic poetry technique.

🧠 **OverTheWire Bandit — All 33 Levels**
Git forensics, restricted shell escapes, vim-based privilege escalation.

🔍 **Real-World Network Assessment**
Independent security assessment on live institutional network — found exposed 
MySQL/PostgreSQL, SSH on port 2222, plaintext FTP/POP3/IMAP. Professional 
responsible disclosure report submitted.

🕵️ **Incident Response — LinkedIn Account Hijacking**
Traced attacker to Nagpur, Maharashtra via email header analysis. Full forensic 
writeup published on Medium.

🛠️ **JARVIS — Local Terminal AI Assistant**
